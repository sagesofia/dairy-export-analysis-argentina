# Argentine Dairy Export Analysis

## Overview

This project analyzes the performance of Argentina's dairy exports between 1990 and 2024 through an interactive Power BI dashboard. The objective is to identify long-term trends in export value, export volume, and implicit export prices using official trade statistics.

## Objectives

* Analyze the evolution of Argentina's dairy exports.
* Compare export value and export volume over time.
* Examine changes in the implicit export price.
* Present key performance indicators (KPIs) through an interactive dashboard.

## Tools

* Power BI
* Microsoft Excel
* Data Analysis
* Data Visualization

## Data Source

* National Institute of Statistics and Census (INDEC)
* Secretariat of Agriculture, Livestock and Fisheries of Argentina

## Dashboard Features

* Export value trend
* Export volume trend
* Export value and volume growth indexes
* Implicit export price by year
* Key performance indicators (KPIs)
* Business insight summarizing the main findings

## Key Insight

Although export volume increased over the analyzed period, export value grew at a significantly faster pace. This suggests that changes in export prices and product composition played an important role in Argentina's dairy export performance.
